## C Programming [ Handwritten Code | Printed Output]

### Functions 

- Write a program to check whether a given number is a prime number using a function.
    ```c
    int isPrime(int number);
    ```
    >The function should return ``1`` if the number is prime, ``0`` otherwise.

- Write a program to calculate the factorial of a number using iteration and a function.
    ```c
    int factorial(int n);
    ```

- Write a program to check whether a given number is a palindrome using a function.
    ```c
    int isPalindrome(int n);
    ```
    >The function should return ``1`` if the number is palindrome, ``0`` otherwise.

- Write a program to display all Armstrong numbers up to a given limit N using a function.
    ```c
    void armstrongUpToN(int n);
    ```
