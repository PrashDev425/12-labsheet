- Write a program to check whether a given number is a prime number using a function.
    ```c
    int isPrime(int number);
    ```
    >The function should return ``1`` if the number is prime, ``0`` otherwise.

- Write a program to calculate the factorial of a number using iteration and a function.
    ```c
    int factorial(int n);
    ```

- Write a program to check whether a given number is a palindrome using a function.
    ```c
    int isPalindrome(int n);
    ```
    >The function should return ``1`` if the number is palindrome, ``0`` otherwise.

- Write a program to display all Armstrong numbers up to a given limit N using a function.
    ```c
    void armstrongUpToN(int n);
    ```

> Input & Output *(Using `alert()`, `prompt()`, `confirm()` and `document.write()` / `document.writeln()`)*

- Write a program to accept two numbers using `prompt()` and display their sum, difference, product, and quotient.
    ```javascript
    function sum(a, b);
    function difference(a, b);
    function product(a, b);
    function quotient(a, b);
    ```

- Write a JavaScript program to check whether a given number is even or odd using an if–else statement.
    ```javascript
    function isEven(number);
    ```
    > The function should return `true` if even, `false` otherwise.

- Write a program to find the largest of three numbers using an if–else-if ladder.
    ```javascript
    function findLargest(a, b, c);
    ```

- Write a JavaScript program to display numbers from 1 to N using a for loop.
    ```javascript
    function displayNumbers(n);
    ```

- Write a program to calculate the factorial of a number using a while loop.
    ```javascript
    function factorial(n);
    ```

- Write a JavaScript program to check whether a number is positive, negative, or zero using a function.
    ```javascript
    function checkNumber(num);
    ```
    The function should return:

    - ``1`` if the number is positive

    - ``-1`` if the number is negative

    - ``0`` if the number is zero

- Write a JavaScript program to perform addition or subtraction based on user choice using switch–case.
    ```javascript
    function calculator(choice, a, b);
    ```
    
    Print Menu using prompt:

    ```js
    let choice = Number(prompt(`
    1) Add
    2) Subtract
    `));
    ```

- Write a JavaScript program to accept **Cost Price (CP)** and **Selling Price (SP)** using `prompt()`. Use `confirm()` to decide the output as follows:  

  - If the user clicks **OK**, display **Profit or Loss along with the amount**. 
  - If the user clicks **Cancel**, display **only Profit or Loss**.

  Display the output using ``document.write()``.

    ```javascript
    function showProfitOrLoss(cp, sp, showAmount);
    ```
